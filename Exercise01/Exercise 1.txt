Exercise 1

주관적으로 답으로만 작성한 문서입니다.

1. 
C언어 : 소스파일 확장자 .c, 컴파일 후 .obj 파일 생성
Java언어: 소스파일 확장자 .java, 컴파일 후 .class 파일 생성

2.
Java 언어를 처음 개발한 사람은 제임스 고슬링(James A. Gosling)

3.
언어개발 순 C, C++, Java, C#

4.
기존의 프로그래밍 언어로 작성된 플랫폼간 호환성이 없어 사용자 컴퓨터에 따라 다시 소스를 컴파일하거나
프로그램을 재작성해야 하는 단점과 가전제품 내부에 내장형 시스템(embedded system)의 메모리 사양이 매우 작아 
플랫폼 독립적(platform independent)이며 메모리 사용량이 적은 새로운 언어를 개발

5.
개발자가 작성한 자바 프로그램을 직접 실행하는 것은 JVM(Java Virtual Machine)인 자바가상기계 이다.

6.
자바는 한 번 작성하면 하드웨어나 운영체제에 상관없이 어디에서나 실행 가능하다. 이를 WORA(Write Once Run Anywhere)라고 한다.

7.
하나의 클래스 파일에는 여러 개의 컴파일된 클래스를 저장할 수 없다.
😰 C언어를 접하지 못해 '클래스가 들어있는 클래스 파일을 찾기 쉽게 하였다'는 의미를 모르겠다. 

8.
JDK(Java Development Kit)으로 자바 응용프로그램을 개발할 수 있다.

9.
클래스 내부에만 전역함수와 전역변수를 둘 수 있다.

10.
Shape.java

11.
A.java. public속성 가진 클래스의 이름으로 저장된다는 의미가 중요한 듯 하다.
/ A.class, 😰 A$B.class, C.class, C$D.class

12.
플랫폼 독립적(platform independent)이라함은 기능의 향상이라기보다 새로운 체계에 가까운 설명이다.
